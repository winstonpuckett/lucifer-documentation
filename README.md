# lucifer-documentation

The public facing documentation website for the [lucifer](https://github.com/winstonpuckett/lucifer) project.

# Getting started

Prerequisites: 
- [node/npm](https://nodejs.org/en/)
- [eleventy](https://www.11ty.dev/)

Steps:
1. Clone the repository
1. cd into the repository root.
1. Run<code>npm install</code>.
1. Run <code>npx @11ty/eleventy --serve</code>.
1. Visit https://localhost:8080 to view the site.
1. Make changes to the source files.

To learn about 11ty, the SSG used in this project, visit [eleventy](https://www.11ty.dev/).
